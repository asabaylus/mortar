Hero With Two Rails Content Package

The Hero With Two Rails content package can accept its data either from an endpoint accessed via AJAX or by having the model embedded directly in the Pestle component. If both sources of data exist, the endpoint will be the winner.

NOTE: because of this component's unusual structure (where it requires DOM elements to be preserved by React, to support child promo authoring), some of its configurations are not testable here. This incudes:

  • Heading Position
  • Disabling Heading Parallax
  • Disabling Rails Parallax

These properties can be thoroughly vetted in AEM
