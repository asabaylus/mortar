Video Modal

Video Modal is the component which loads the video in the same page using a popup window. An example can be found by clicking on the video promo card.

To use this component it must be present somewhere on the page.

        
          <div data-pestle-module="VideoModal" />

Create a function to call within the component you are launching the modal from and pass the data through Pestle's Pub/Sub (don't forget to bind it in your constructor)

        
          constructor(props) {
            super(props);
            this.launchModal = this.launchModal.bind(this);
          }

          launchModal(){
            const videoData = {
              "itemId" : this.props.itemId,
              "leadMedia" : this.props.leadMedia,
              "text" : this.props.text
            }
            Pestle.PubSub.publish(events.launchVideoModal, videoData);
          }

Attach the function to the onClick event of the element you want to launch it from.

        
          <div onClick={this.launchModal} />
