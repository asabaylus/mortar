Install Mortar with npm

Current version of Mortar (v3) is currently under development. Be aware that you're using software that is in development and may contain errors or bugs.

If you want a part of Mortar, use our private npm account (@natgeo).

$ npm install @natgeo/mortar_v3

Choose just the component you want to use and require/import it into your code. ex:

import HeroWithTwoRailsComponent from '@natgeo/mortar_v3/lib/contentPackages/heroWithTwoRails/heroWithTwoRails';

Using Mortar SCSS/CSS

Mortar's SCSS and CSS are included in the package.
Use precompiled bundle if:

- You want start developing a.s.a.p.
- You don't want to add task to compile sass files.
- You don't want to worry about digging sass files.
- You're ok adding all mortar's css on your project.

# Precomiled CSS
node_modules/@natgeo/mortar_v3/lib/styles/mortar-default.min.css

Use scss files if you want:

- Use styles for specific components of Mortar
- Keep css bundle smaller
- Keep control on the css you're including.
You'll find all scss file on node_modules/@natgeo/mortar_v3/lib/styles/

# Layout SCSS
node_modules/@natgeo/mortar_v3/lib/styles/grid/_rows.scss
node_modules/@natgeo/mortar_v3/lib/styles/grid/_columns.scss

Using Mortar SVG Icons

Mortar SVG icons are also included on the package.
Use SVG sprite if:
- You want to use good amount of icons from mortar
- You're using a lot of components from mortar that includes svg icons

To import sprite SVG on your project, just add all the content from the following svg file below the opening body tag definition (<body>).

node_modules/@natgeo/mortar_v3/lib/icons/mortar-symbol-sprite.svg

Use individual SVG icons if:
- Specific icons are required
- You are not using mortar components
- You're creating your own svg sprite

Individual icons can be found:

node_modules/@natgeo/mortar_v3/src/icons/

Using Mortar Fonts

Mortar's fonts are available when you install the main CSS file in your site. Due to license restrictions fonts will not load outside a *.nationalgeographic.com domain. During development modify your etc host file as follows: 127.0.0.1 yourAppHere.nationalgeographic.com
