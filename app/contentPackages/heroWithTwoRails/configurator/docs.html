Hero With Two Rails Content Package

The Hero With Two Rails content package can accept its data either from an endpoint accessed via AJAX or by having the model embedded directly in the Pestle component. If both sources of data exist, the endpoint will be the winner.

The below configurator only allows manipulation of the 2 component-wide properties - not the child Promos, which will be individually editable when used in CMS environments such as AEM. Neither configurable property is very impactful at the moment, pre-styling work.
