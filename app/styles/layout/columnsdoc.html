Reorder: The “push” and “pull” classes can be used to switch the graphical presentation from the source order of the HTML.

 
 
{% include "./_partials/grid.html" %} 
<div class="mt_cf">
  <div class="mt_col-3 mt_col-push-9 mt_col-gut-sm mt_col-md-gut-md mt_col-lg-gut-lg">
    <div class="exampleborder mt_row-gut-1"> </div>
  </div>

  <div class="mt_col-9 mt_col-pull-3 mt_col-gut-sm mt_col-md-gut-md mt_col-lg-gut-lg">
    <div class="exampleborder mt_row-gut-1"> </div>
