Column: This property refers to the number of columns a given object spans. Values range from 1 to 12.

 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

Responsive Grid: Use the breakpoint classes to create an object that spans 6 columns on mobile, or half the page width, and 3 columns on desktop, or a quarter of the page width, for example.

 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

Offset: This property is used to create empty columns of space in between objects. Breakpoint classes are also available for this property.

 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

Reorder: The “push” and “pull” classes can be used to switch the graphical presentation from the source order of the HTML.
