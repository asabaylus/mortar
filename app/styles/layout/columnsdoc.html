Columns: The classes provided are based upon a twelve column grid available at any breakpoint

 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

Responsive Grid: Use the breakpoint classes to override layouts at certain browser widths. Mobile first!

 
 
 
 
 
 
 
 
 
 
 
 

Offsetting: Offset classes use `margin-left` to "skip" columns and create larger gaps between rows.

 
 
 
 

Reorder: "Push" and "pull" classes can be used to switch the graphical presentation from the source order of the html.
