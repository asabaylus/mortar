Page Headings and Descriptions

.mt_h1

Magazine

This class should be used for very short page titles, often used on top-level hub pages.

.mt_h2

This Is an Article Headline, Around 60 Characters Long

This class should be used for longer page titles, such as article headlines.

.mt_h3

Component Header

This style is used as a heading for smaller components on the page, such as groups of promos, maps and graphics, or in-line photo galleries.
