Using the Color Class Names

The color classes on this page may be applied to an object's background, border, or fill.

To alter an object's background color, use the mt_bgcolor--* class. Replace the asterisk with the desired color keyword, like primary, shown below. See the bottom of this page for more keywords.

 
<div class="mt_bgcolor--primary"></div>

To alter an object's border color, use the mt_border--* class.

 
<div class="mt_bordercolor--primary"></div>

To alter text color or an SVG's fill color, use the mt_color--* class.
