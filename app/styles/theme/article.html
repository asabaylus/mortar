Articles and Inline Styles

Inside of a wrapping element with the .mt_article class the following elements can be used without specifying a class. While most of mortar avoids relying on the cascade, this class relies upon *nested selectors* so it can work with the output of content generated from WYSIWYGs.

Block quotes are used to offset text that has been quoted or excerpted from another source. It is typically used for longer blocks of text that are too long to display inline. The source of the quote should always be attributed when possible.
