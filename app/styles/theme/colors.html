Primary Colors

If you want to use a light variation of the primary color, your class name will be .mt_primary-light

To use the danger color theme, use .mt_text-color-danger for text, and .mt_bg-color-danger for background colors.

Here's a list of class names you can use for primary and neutral background colors:

Primary Background colors

.mt_bg-color-primary

Neutral Background Colors / Light

.mt_bg-color-neutral

You can also use the following modifiers. Source: BEM

.mt_bg-color-neutral--light

.mt_bg-color-neutral--lighter

.mt_bg-color-neutral--lighter-even

Neutral Background Colors / Dark

.mt_bg-color-neutral--dark

.mt_bg-color-neutral--darker

.mt_bg-color-neutral--darkest
